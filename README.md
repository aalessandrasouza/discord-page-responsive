# Projet Discord - Défi de Recréation Responsive

Ce projet consiste à reproduire une page inspirée de la plateforme Discord, utilisant HTML et CSS de manière responsive, sémantique et avec un code de qualité.

---

## À propos du projet

La page recrée les principaux concepts visuels et fonctionnels de Discord, en mettant l'accent sur :

- Une interface accessible et organisée en différentes sections.
- La communication dans des serveurs avec des canaux texte et voix.
- Des outils de modération et de gestion des rôles.
- Les appels vidéo et le partage d'écran à faible latence.
- Un design responsive pour s'adapter à différents appareils.

---

## Structure de la page

1. **Header Hero**  
   - Message de bienvenue et invitation à télécharger ou ouvrir dans le navigateur.  
   - Illustration mettant en avant le concept de communauté.

2. **Sections de contenu**  
   - **Créez un espace contrôlé par invitation où vous vous sentez chez vous** : Explique les canaux texte et l’interaction entre membres.  
   - **Ici, il est facile de se retrouver** : Montre les canaux vocaux et l’interaction instantanée.  
   - **Pour petits et grands groupes** : Explication des outils de modération et de l’accès personnalisé.  
   - **Technologie de connexion fiable** : Met en avant la qualité des appels voix et vidéo.

3. **Footer**  
   - Logo Discord pour clôturer visuellement la page.

---

## Technologies utilisées

- **HTML5** pour la structure sémantique de la page.  
- **CSS3** avec Flexbox pour un layout responsive.  
- **Google Fonts** : 'Luckiest Guy' pour les titres et 'Roboto' pour les textes.  
- Images et icônes téléchargées depuis Figma au format PNG.

---

## Comment utiliser

1. Clonez ou téléchargez le dépôt.  
2. Ajoutez les images PNG téléchargées depuis Figma dans les dossiers indiqués dans le code HTML (`src` des balises `<img>`).  
3. Ouvrez le fichier `index.html` dans votre navigateur pour voir la page.  
4. Pour personnaliser, modifiez les fichiers `index.html` et `styles.css` à votre convenance.

---

## Responsive

- Le layout s’adapte à différentes tailles d’écran, du desktop au mobile.  
- Les boutons et textes s'adaptent pour garantir une bonne expérience utilisateur.  
- Les images et illustrations conservent des proportions et un niveau de détail accessibles.
